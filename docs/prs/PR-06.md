# PR #06 — Onboarding guiado de 3 pasos (conectar → definir cola → primer post)

**Área:** UX
**Prioridad:** 🟡 Importante
**Esfuerzo estimado:** M (2-8h)

**Problema:**
No hay onboarding. El dashboard vacío no dice qué hacer. La activación depende de conectar cuenta (`/integrations`) y definir slots (`/queue`), pero nada lo guía.

**Solución propuesta:**
Banner/checklist en el dashboard que detecte estado real: `(1) ¿tiene integrations? (2) ¿tiene queueSlots? (3) ¿tiene ≥1 post?`. Mostrar 3 pasos con CTA directo a `/integrations`, `/queue`, `/posts/new`. Ocultarlo al completarse. Los datos ya están disponibles vía endpoints existentes.

**Archivos a modificar:**
- `apps/web/src/app/page.tsx` (o nuevo `apps/web/src/components/OnboardingChecklist.tsx`)

**Criterios de aceptación:**
- [ ] El checklist refleja estado real y desaparece al completar los 3 pasos
- [ ] Cada paso enlaza a la pantalla correspondiente
