# PR #18 — Convertir guion viral → post programado (Brain ↔ calendario)

**Área:** UX / Feature
**Prioridad:** 🟢 Mejora
**Esfuerzo estimado:** M (2-8h)

**Problema:**
Un `GeneratedScript` no se puede enviar a la cola/calendario en un paso; hay que copiar el contenido a mano, y el vínculo `GeneratedScript.postId` (que alimenta el feedback loop) casi nunca se establece.

**Solución propuesta:**
Botón "Programar este guion" en `/brain` que precargue `/posts/new` con `hook+body+cta+hashtags` como caption (querystring o estado compartido) y que al crear el post llame a `markPublished(scriptId, postId)` automáticamente para cerrar el feedback loop.

**Archivos a modificar:**
- `apps/web/src/app/brain/page.tsx`
- `apps/web/src/app/posts/new/page.tsx` (aceptar prefill)
- `apps/api/src/modules/brain/brain.controller.ts` (si hace falta endpoint de vínculo)

**Criterios de aceptación:**
- [ ] Prefill de caption desde un guion en un click
- [ ] `GeneratedScript.postId` se setea al publicar → métricas se recogen solas a las 72h
