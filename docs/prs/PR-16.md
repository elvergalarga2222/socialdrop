# PR #16 — Feedback loop: métricas multi-plataforma + accuracy empírico

**Área:** Feature (Content Brain)
**Prioridad:** 🟢 Mejora
**Esfuerzo estimado:** L (1-2 días)

**Problema:**
`BrainService.collectMetricsForScript` solo implementa Instagram; los guiones publicados en FB/YT nunca reciben métricas → nunca marcan `isViral` → el brain aprende sesgado a IG. Además `accuracyScore` lo **inventa el LLM** en el prompt de aprendizaje, no se valida contra resultados reales.

**Solución propuesta:**
1. Implementar recolección para FACEBOOK y YOUTUBE reutilizando `MetricsService` (likes/comments/shares FB; views/likes YT).
2. Calcular `accuracyScore` empíricamente: % de guiones generados tras el último aprendizaje que superaron el `avgEngagement` histórico del usuario.
3. Enviar a Claude también `body`/`cta` y ponderar los ejemplos por engagement real.

**Archivos a modificar:**
- `apps/api/src/modules/brain/brain.service.ts`

**Criterios de aceptación:**
- [ ] `isViral` se evalúa para IG/FB/YT
- [ ] `accuracyScore` derivado de datos, no del LLM
